CSS

Tutorial from csstutorial.net

Syntax: Selector {property:value;}

Selectors:

  • element: p
  • class: .mw-some-class
  • id: #some_id
  • element and class: div.some_class

Stylesheet priority: 1. Inline, 2. Style Sheet, 3. Browser default

ID's: IDs should be unique and can be referenced with css.

padding: 10px 10px 10px 10px /*top right bottom left, wrap arround 10 pixels on each site.*/

Marging vs. Padding: Marging is outside, Padding is inside

marging auto: horizontally center an element

position: absolute;

Combinators: Selectors can be combined in the following ways:

  • descendants: p div
  • child: p > div
  • adjacent siblings: div + p
  • general siblings: div ~ p

Preudo classes: Pseudo classes

SASS

the sass documentation

In sass stuff can be nested. i.e. .a { .b {width:10px;}} translates to .a .b {width:10px;}

can be use for pseudo classes: i.e. a { &:hover {color: red;}} translates to a:hover {color: red;}

nested propertis: a: {b{color:red;}} translates to a-b {color:red;}
